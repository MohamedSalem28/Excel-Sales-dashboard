# 📊 Excel Sales Dashboard Analysis

## 📌 Overview
This project analyzes sales data using Excel to extract meaningful business insights, track performance, and understand customer behavior.

## 🛠️ Tools Used
- Microsoft Excel

## 📈 Key Metrics
- Total Revenue
- Total Orders
- Average Delivery Time
- Average Customer Spend

## 📊 Analysis Performed
- Revenue by Month
- Revenue by Product
- Revenue by Occasion
- Orders by Location
- Revenue by Category
- Revenue by Hour

## 🔍 Key Insights
- Sales performance varies significantly across months, indicating seasonal trends.
- Certain products contribute the highest share of revenue.
- Customer behavior changes based on time of day and location.
- Occasion-based analysis helps identify peak demand periods.
